# alkhaleejarestaurant



\# 🍴 Al-Khaleej Arabian Restaurant Management System



A complete web-based restaurant management system with customer ordering, admin dashboard, and staff portal.



\## ✨ Features



\### 👤 Customer Portal

\- Browse menu without registration

\- Add items to cart

\- Place orders by table number

\- Track order status

\- Submit feedback



\### 🔐 Admin Panel

\- Dashboard with analytics

\- Manage menu items (Add/Edit/Delete)

\- Staff management

\- View and update order status

\- Generate reports



\### 👔 Staff Portal

\- View pending orders

\- Update order status

\- Daily order tracking



\## 🛠️ Technologies Used



\- \*\*Frontend\*\*: HTML, CSS, JavaScript

\- \*\*Backend\*\*: PHP

\- \*\*Database\*\*: MySQL

\- \*\*Server\*\*: Apache (XAMPP/WAMP/LAMP)



\## 📋 Prerequisites



\- XAMPP/WAMP/LAMP installed

\- MySQL database

\- Web browser
